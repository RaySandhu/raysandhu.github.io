Projects

An intersection of my healthcare background and technical knowledge, I built this desktop application in Java as a tool for users to adhere to medication schedules. While addressing an issue that every healthcare worker has noted in their patients, this project was an exercise in best practices for object-oriented programming.

Part of a larger project that is under construction, I made this Flappy Bird derivative game as a two-pronged introduction to game development and asset design.
